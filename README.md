# Steel Pans

Interactive steel pan practice tool built with Blazor. [Try it out here.](https://pans.nmayes.dev)

## Features

- Play steel pan layouts directly in the browser
- MIDI playback with visual note highlighting
- Built-in metronome with adjustable tempo and time signature
- Chord builder for exploring harmonies
- Responsive UI for desktop and mobile

## Tech Stack

- Blazor (.NET)
- JavaScript (Web Audio API)
- DryWetMIDI for MIDI processing
